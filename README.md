# 🐾 Adopt Me - Pet Adoption Website

A modern and responsive pet adoption website built using HTML and CSS.

## 🌟 Features

* Responsive design for mobile and desktop
* Modern card-based layout
* Beautiful pet profiles
* Hover animations and smooth transitions
* Attractive UI with custom styling
* Fully static website hosted on GitHub Pages

## 📸 Preview

This website allows users to browse pets available for adoption and learn more about them.

### Pets Featured

* 🐶 Jerry – Friendly and loyal dog
* 🐱 Tom – Affectionate and playful cat

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
